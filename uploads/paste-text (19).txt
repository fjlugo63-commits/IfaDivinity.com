### 2. Cross‑Module Integration

#### 2.1 Dashboard Integration
Dashboard must link to:
- Booking Flow (CP‑2)
- Consultation History (CP‑4)
- Payments (CP‑3)
- Messages (CP‑5)
- Profile (CP‑6)

#### 2.2 Booking Flow Integration
Booking Flow must:
- Redirect accepted bookings to Consultation Detail (CP‑4)
- Redirect accepted bookings to Payment Status (CP‑3)
- Show booking status on Dashboard

#### 2.3 Payments Integration
Payments must:
- Link back to Consultation Detail (CP‑4)
- Link to Ebo detail (if applicable)
- Show receipts
- Update Dashboard payment cards

#### 2.4 Consultation History Integration
Consultation Detail must:
- Link to Payments (CP‑3)
- Link to Ebo (if applicable)
- Show summary viewer
- Show booking reference

#### 2.5 Messaging Integration
Messaging must:
- Show unread count in navigation bar
- Link from Dashboard
- Link from Consultation Detail (optional)

#### 2.6 Profile Integration
Profile must:
- Update timezone across all modules
- Update name/email on Dashboard
- Update timestamps in:
  - Booking Flow
  - Consultation History
  - Payments
