# Project Summary
The Ifa Divinity Marketplace is an online platform that connects practitioners of Ifa divination with clients seeking spiritual readings and services. It enhances user experience through seamless browsing, booking, and real-time data access via Supabase, reflecting the rich cultural nuances of Yoruba traditions. The platform now features an **Awo Dashboard** for practitioners, which includes a new **Consultation Workspace** that allows manual entry of Odu selections, streamlining the consultation process.

# Project Module Description
The project consists of two main modules:
- **Frontend**: A React-based interface for users to explore services, manage bookings, and authenticate accounts. It leverages Supabase for real-time updates and includes functionalities for handling Egbo services, the Awo Dashboard, and the new Consultation Workspace for Manual Odu Entry.
- **Backend**: Manages database interactions and business logic using Supabase, efficiently handling user data and session states. Supports Egbo service management, Stripe payment processing, and the Awo Dashboard API, along with new functionalities for Odu operations.

# File Description Inventory
- **README.md**: Documentation for both backend and frontend modules.
- **skills_docs/**: Documentation for Supabase integration and requirements.
- **src/**: Contains all source code for the frontend application, including components, pages, hooks, and contexts.
- **lib/audit.ts**: Implements audit logging functionality for tracking significant actions across the marketplace.
- **lib/supabase.ts**: Configures and exports the Supabase client with centralized table name mappings, including new tables for the Awo Dashboard and Odu operations.
- **contexts/AuthContext.tsx**: Manages user authentication and session states with real Supabase data.
- **pages/ConsultationWorkspace.tsx**: New page providing Awo practitioners with a workspace for manual Odu entry, including Odu selection and confirmation processes.
- **pages/Admin.tsx**: Admin panel for managing users, orders, products, and seller verification.
- **pages/SellerDashboard.tsx**: Dashboard for sellers to manage their products and orders.
- **pages/Cart.tsx**: Shopping cart interface with checkout functionality, including Egbo service integration.
- **pages/SuperAdminSetup.tsx**: Updated page for creating or elevating accounts to Super Admin, now with improved error handling and detailed error messages.
- **supabase/functions/app_odu_entry/index.ts**: Handles API endpoints for Odu operations related to manual entry.
- **e2e/egbo-flow.spec.ts**: Playwright E2E test for validating the complete Egbo service flow.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Backend**: Supabase (PostgreSQL, Authentication, Edge Functions)
- **Payment Integration**: Stripe
- **Development Tools**: ESLint, PostCSS, Playwright

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Run the application:
   ```
   pnpm run start
   ```
