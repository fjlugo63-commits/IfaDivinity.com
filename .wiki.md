# Project Summary
The Ifa Divinity Marketplace is an online platform designed to connect practitioners of Ifa divination with clients seeking spiritual readings and related services. This marketplace enhances user experience by providing seamless browsing, booking, and real-time data access through Supabase, all while embodying the rich cultural nuances of Yoruba traditions.

# Project Module Description
The project consists of two main modules:
- **Frontend**: A React-based interface that allows users to explore services, manage bookings, and authenticate accounts. It leverages Supabase for real-time data updates.
- **Backend**: Manages database interactions and business logic using Supabase, handling user data and session states efficiently.

# File Description Inventory
- **README.md**: Documentation for both backend and frontend modules.
- **skills_docs/**: Documentation for Supabase integration and requirements.
- **src/**: Contains all source code for the frontend application, including components, pages, hooks, and contexts.
- **lib/audit.ts**: Implements audit logging functionality for tracking significant actions across the marketplace.
- **lib/supabase.ts**: Configures and exports the Supabase client with centralized table name mappings.
- **contexts/AuthContext.tsx**: Manages user authentication and session states with real Supabase data.
- **pages/Bookings.tsx**: Displays demo practitioners when no real profiles are available.
- **components/ImageUpload.tsx**: Facilitates image uploads for sellers, allowing multiple images per product.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Backend**: Supabase (PostgreSQL, Authentication)
- **Development Tools**: ESLint, PostCSS

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Run the application:
   ```
   pnpm run start
   ```
