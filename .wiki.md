# Project Summary
The Ifa Divinity Marketplace is an online platform that connects practitioners of Ifa divination with clients seeking spiritual readings and related services. It provides a seamless user experience for browsing and booking sessions, while fostering an inviting atmosphere inspired by Yoruba culture.

# Project Module Description
The project features two primary modules:
- **Frontend**: A React-based interface enabling users to browse services, manage bookings, and authenticate accounts. It now integrates with real Supabase data.
- **Backend**: Handles database interactions and business logic using Supabase to manage user data and session states.

# File Description Inventory
- **README.md**: Documentation for both backend and frontend modules.
- **skills_docs/**: Contains documentation for Supabase integration and requirements.
- **src/**: Contains all source code for the frontend application, including components, pages, hooks, and contexts.
- **lib/supabase.ts**: Configures and exports the Supabase client with a centralized table names mapping.
- **contexts/AuthContext.tsx**: Manages user authentication and session states with real Supabase data.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Backend**: Supabase (PostgreSQL, Authentication)
- **Development Tools**: ESLint, PostCSS

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Run the application:
   ```
   pnpm run start
   ```
