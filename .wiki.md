# Project Summary
The Ifa Divinity Marketplace is an online platform designed for spiritual readings and related services. It connects practitioners with clients seeking guidance and insights, featuring a user-friendly interface for browsing services, booking sessions, and managing orders. The marketplace emphasizes a warm, earth-tone aesthetic inspired by Yoruba culture, creating an inviting atmosphere for users.

# Project Module Description
The project consists of two main modules:
- **Frontend**: A React-based user interface that allows users to interact with the marketplace, including browsing products, managing bookings, and authenticating accounts.
- **Backend**: Responsible for database interactions and business logic, utilizing Supabase for managing user data and session states.

# File Description Inventory
- **README.md**: Documentation for both backend and frontend modules.
- **skills_docs/**: Contains documentation for Supabase integration and requirements.
- **src/**: Contains all source code for the frontend application, including components, pages, hooks, and contexts.
- **lib/supabase.ts**: Configures and exports the Supabase client, handling environment variables gracefully.
- **contexts/AuthContext.tsx**: Manages user authentication and session states.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Backend**: Supabase (PostgreSQL, Authentication)
- **Development Tools**: ESLint, PostCSS

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Run the application:
   ```
   pnpm run start
   ```
