# Project Summary
The Ifa Divinity Marketplace is an online platform designed to connect practitioners of Ifa divination with clients seeking spiritual readings and related services. It enhances user experience through seamless browsing, booking, and real-time data access via Supabase, while reflecting the rich cultural nuances of Yoruba traditions. Recent updates have introduced features for managing Egbo services, including server-side booking management and enhanced payment integration through Stripe.

# Project Module Description
The project consists of two main modules:
- **Frontend**: A React-based interface that allows users to explore services, manage bookings, and authenticate accounts. It leverages Supabase for real-time data updates and includes functionalities for handling Egbo services.
- **Backend**: Manages database interactions and business logic using Supabase, handling user data and session states efficiently, with added support for Egbo service management and Stripe payment processing.

# File Description Inventory
- **README.md**: Documentation for both backend and frontend modules.
- **skills_docs/**: Documentation for Supabase integration and requirements.
- **src/**: Contains all source code for the frontend application, including components, pages, hooks, and contexts.
- **lib/audit.ts**: Implements audit logging functionality for tracking significant actions across the marketplace.
- **lib/supabase.ts**: Configures and exports the Supabase client with centralized table name mappings.
- **contexts/AuthContext.tsx**: Manages user authentication and session states with real Supabase data.
- **pages/Bookings.tsx**: Displays demo practitioners when no real profiles are available.
- **components/ImageUpload.tsx**: Facilitates image uploads for sellers, allowing multiple images per product.
- **pages/Admin.tsx**: Admin panel for managing users, orders, products, and seller verification.
- **pages/SellerDashboard.tsx**: Dashboard for sellers to manage their products and orders.
- **pages/Cart.tsx**: Shopping cart interface with checkout functionality, including Egbo service integration.
- **supabase/functions/app_egbo_checkout/index.ts**: Handles checkout session creation and order processing for Egbo services.
- **supabase/functions/app_egbo_webhook/index.ts**: Processes Stripe webhook events for payment confirmation and booking finalization.
- **e2e/egbo-flow.spec.ts**: Playwright E2E test for validating the complete Egbo service flow.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Backend**: Supabase (PostgreSQL, Authentication, Edge Functions)
- **Payment Integration**: Stripe
- **Development Tools**: ESLint, PostCSS, Playwright

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Run the application:
   ```
   pnpm run start
   ```
